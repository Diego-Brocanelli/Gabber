BuilderGen é um gerador de versões para projetos de software. Seu uso é simples, fácil e é dedicado para projetos pequenos.

Download 1.0.6

Qualquer projeto pode utilizar o BuilderGen

BuilderGen gera números sequenciais para versões.

Por exemplo, em builds da Microsoft você sabe que a versão 1109 é bem anterior à 14563. Contudo, a 14573 tem poucas diferenças para a 14563.

Requisitos

  • Java na versão 1.8
  • Linux
  • Não testado em Mac

Recomendações

Recomendamos que o projeto utilize Git, Composer e Linux.

Formatação

O formato atual das versões é 1.0.{build}-{yyyyMMdd}_{HHmmss}.

O suporte a edição do formato será liberado em breve.
